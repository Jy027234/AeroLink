# P2-04 后端模块化单体首批

日期：2026-07-19（Asia/Shanghai）
状态：首批完成（四个模块的 Repository/Policy/Service 入口与报价→订单主链跨聚合用例已落地；报价发送队列也已收进聚合 service，剩余为更细依赖图约束和非首批基础设施适配）

## 已交付

- 新增 `server/src/modules/quotationOrder` 与 `server/src/modules/rfqSourcing` 公开模块入口，将报价状态策略、UI 投影、事务状态服务和 RFQ 状态迁移/规范化作为模块用例边界暴露。
- `quotations`、`rfqs` 路由改从模块入口调用状态策略/事务服务，既有 URL、状态码、幂等、审计和 Outbox 行为保持不变；报价创建（含金额 Decimal 双写、AOG RFQ→QUOTING、审批通知）、提交/审批、发送队列、确认（订单/合同生成或复用）和撤回（预留释放/撤回通知），以及 `/orders` 直接成单已由 `quotationOrder` 聚合 service 负责。
- 新增 `customerSupplier` 与 `inventoryQuality` 公开模块入口；Customer、Supplier、InventoryDetail 及 InventoryTransaction 的路由查询/创建/更新已改走 repository 边界，供应商邮箱/等级、客户状态、库存数量调整等 policy 已移入模块 service；Customer nested update、Supplier email uniqueness/create/update 使用模块事务 service，Inventory receipt/detail update/delete 与 inbound/adjustment ledger，以及 reservation/release/outbound 与订单状态联动，也由 `inventoryQuality` 聚合 service 在同一事务内完成，避免删除/重建关系时留下半完成状态。
- `rfqSourcing`、`quotationOrder` 新增 RFQ/Quotation/Order 查询 repository 入口；报价→订单创建用例从 `quotationOrder` 公共入口暴露，路由不再直接跨到旧 workflow 文件。
- Order 状态迁移与可变字段更新现在由 `quotationOrder` service 负责：service 执行存在性/状态策略、乐观版本条件、权限回调和 `order.status.changed` Outbox；HTTP 路由仅提供参数、幂等上下文和响应投影。
- RFQ 创建、更新和初始状态历史已由 `rfqSourcing` service 承担，路由保留权限、幂等、Outbox 和响应映射；新增服务单测并在隔离 PostgreSQL 上复验 RFQ 主链/幂等/状态治理 E2E 5/5。
- 模块服务单测覆盖合法/非法状态迁移、UI 状态投影、库存收货、数量调整流水、预留写入，以及库存删除前的 legacy/流水/证书保护。
- 新增受数据库名保护的真实 PostgreSQL 集成检查：在隔离 seed 库验证 RFQ 状态历史事务回滚、Inventory policy、Quotation/Order 事务用例与 Customer/Supplier 写入；临时供应商记录在检查后清理。
- 模块化后在隔离 PostgreSQL/API/前端环境重跑报价→订单契约 E2E 9/9，覆盖成单/合同下载及报价、订单详情、发送、撤回和 PDF 失败恢复路径。
- `server/src/index.ts` 增加主模块判定：被测试/其他模块导入时只构造 `app/httpServer/io`，不监听端口、不启动内嵌 Worker；直接执行时才启动 HTTP 生命周期。

## 未完成

Inventory、Customer/Supplier 的直接 repository 与首批 policy 收口已完成；库存创建/调整/删除/预留/释放/出库聚合写入，以及报价创建/AOG 联动、提交/审批、发送队列、确认/撤回、订单直接成单、订单状态迁移和可变字段更新已移入 service。剩余为其他基础设施适配和更细的依赖图约束，将在下一批按交易链顺序推进。当前仍保留原路由作为薄 HTTP 适配层。`npm run modules:check` 已对四个公开模块入口执行跨边界检查；库存 canonical receipt/adjustment 与读路径在隔离 PostgreSQL 上的 Playwright 回归为 2/2，RFQ→报价→订单→库存主链定向回归为 3/3，浏览器预留与幂等回归为 1/1。
